Portfolio – Gabriel Durand‑Touya  

Bienvenue sur le portfolio personnel de **Gabriel Durand‑Touya**, un projet Web simple mais soigné pour mettre en valeur mes compétences, mes projets et ma personnalité.

---

## 🇫🇷 Version Française  

### 🎯 Objectif  
Ce site a pour but de présenter :  
- Qui je suis : ma formation (pré‑MSc informatique à EPITECH Toulouse, mes passions pour les jeux vidéo et l’informatique), où j’en suis, mes objectifs.  
- Mes projets techniques : développements réalisés, code hébergé, démonstrations.  
- Les moyens de me contacter (réseaux, mail).  
- Une vitrine professionnelle/adaptée à mes ambitions d’alternance et de carrière.  

### 🛠 Technologies  
Le site est construit en :  
- HTML & CSS pour la structure et le style.  
- JavaScript (vanilla) pour quelques interactions simples.  
- Structure statique, conception responsive (adaptée mobile/desktop).  
- Aucun framework lourd pour garder la simplicité, la rapidité et la lisibilité.  

### 📂 Organisation des fichiers  
```
/css       → feuilles de style (ex : style.css)  
/html      → fichiers HTML (ex : index.html)  
/js        → scripts JavaScript (ex : main.js)  
lm 15.pdf  → document PDF (CV, projet ou rapport)  
```

### 🔍 À découvrir  
- Page d’accueil : présentation, photo/logo, slogan.  
- Section « À propos » : parcours, motivations.  
- Section « Projets » : réalisations et liens.  
- Section « Contact » : formulaire ou liens vers LinkedIn, GitHub, etc.  
- Section « CV » : lien de téléchargement (`lm 15.pdf`).  

### ✅ Utilisation  
1. Clone le dépôt :  
   ```bash
   git clone https://github.com/gabrieldurandtouya/Portfolio.git
   ```  
2. Ouvre `html/index.html` dans ton navigateur.  
3. Modifie les fichiers selon tes besoins.  
4. Pousse tes changements sur GitHub.  
5. (Optionnel) Héberge sur GitHub Pages, Netlify ou Vercel.  

### 🚀 Améliorations envisagées  
- Animations et transitions CSS/JS.
- Section blog ou journal.  
- Optimisation des performances.  


### 📄 Licence  
libre d’utilisation et de modification avec mention de l’auteur.
